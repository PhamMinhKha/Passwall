# INSTALL PASSWALL : Run this command in openwrt remote ssh
```
rm -f passwallx.sh && wget https://raw.githubusercontent.com/PhamMinhKha/Passwall/main/passwallx.sh && chmod 777 passwallx.sh && sh passwallx.sh
```
Done !
